# MITS-ON-2026-JX-B1

This is the code repository of the guided project based industry-aligned experience program.

**JobXperience™** program by **PSNM Innovations** places the students inside job-style execution environments where experience is built through real responsibility, deadlines, and outcomes — supported every step of the way.

This program is run in collaboration with **MITS - Madanapalle Institute of Technology & Science** (https://mits.ac.in/) and **PSNM Innovations** (https://psnm.in/). 

If you would like to be part of similar future training programs, please drop us an email at `hello@psnm.in`.
